- Ubuntu Server 22.04 LTS

## Tools Used
- MobaXterm (SSH Client)
- Linux Terminal

## Connection Method
Connected to AWS EC2 instance using SSH via MobaXterm with PEM key authentication.

## Linux Commands Practiced

### Navigation Commands
- `pwd` – Displays the current working directory
- `ls` – Lists files and folders
- `cd ~` – Moves to home directory
- `cd /` – Moves to root directory

### File & Directory Commands
- `mkdir` – Creates directories
- `touch` – Creates files
- `rm`, `rmdir` – Deletes files and folders

### File Viewing & Editing
- `nano` – Edits files
- `cat`, `less` – Views file content

### Permissions & Ownership
- `ls -l` – Shows file permissions
- `chmod` – Changes file permissions
- `chown` – Changes file ownership

### System Monitoring
- `top`, `htop` – Monitors system usage
- `df -h` – Disk usage
- `free -m` – Memory usage


## Key Learnings
- Linux directory structure
- File permissions and ownership
- Secure remote access using SSH
- Basic system monitoring

## Challenges Faced
- Permission changes not reflecting on mounted directories
- Understood the difference between Linux and Windows filesystems

Successfully completed Linux environment setup and gained hands-on Linux experience on AWS EC2.
